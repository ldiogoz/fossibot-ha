# Fossibot Power Station

Home Assistant integration for Fossibot portable power stations via the BSApp cloud API.

**31 entities** across 5 platforms: sensors, switches, number controls, selects, and buttons.

**Energy Dashboard ready** with solar production, charge energy, and consumption tracking.

Supports real-time MQTT updates, auto token refresh, configurable polling, multi-device discovery, device availability detection, diagnostics, and full device control including AC/DC/USB outputs, LED modes, charge limits, standby timers, and remote shutdown.

Tested with Fossibot F2400-B. Other models using the BSApp / BrightEMS app should also work.
